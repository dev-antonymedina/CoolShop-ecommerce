# CoolShop – Ecommerce

**CoolShop** es una aplicación tipo **e-commerce** construida con **React** y **Vite**, pensada para demostrar una tienda online moderna.  

## Demo  
Puedes ver la versión desplegada aquí: *[cool-shop-ecommerce.vercel.app](https://cool-shop-ecommerce.vercel.app)* :contentReference[oaicite:0]{index=0}  

## Tecnologías usadas  
- React  
- Vite  
- ESLint  
- CSS / HTML  

## Instalación

1. Clona el repositorio:  
   ```bash
   git clone https://github.com/dev-antonymedina/CoolShop-ecommerce.git
   
2. Entra al directorio del proyecto:

cd CoolShop-ecommerce


3. Instala las dependencias:

npm install


4. Corre la app en modo desarrollo:

npm run dev
